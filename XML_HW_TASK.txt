XML

1. Создать внешний репозиторий c названием XML.
 https://github.com/ > Sign in > Click "New" on top-left corner> Click "Repository name" tab > Enter XML > Put a check next to "Add a README file" > Click "Create repository".
2. Клонировать репозиторий XML на локальный компьютер.
 git clone https://github.com/ee1ovc/XML.git
3. Внутри локального XML создать файл “new.xml”.
 > new.xml
4. Добавить файл под гит.
 git add new.xml
5. Закоммитить файл.
 git commit -m "Xml"
6. Отправить файл на внешний GitHub репозиторий.
git push
7. Отредактировать содержание файла “new.xml” - написать информацию о себе (ФИО, возраст, количество   домашних животных, будущая желаемая зарплата). Всё написать в формате XML.
vim new.xml
press i
<info>
<name>Ivan</name>
<surname>Ovcharenko</surname>
<age>33</age>
<petCount>0</petCount>
<wantedSalary>55000</wantedSalary>
</info>
8. Отправить изменения на внешний репозиторий.
 git add new.xml
 git commit -m "age,full name,pet count, wanted salary" 
 git push
9. Создать файл preferences.xml
10. В файл preferences.xml добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате XML.
оздать файл sklls.xml добавить информацию о скиллах которые будут изучены на курсе в формате XML
11. Сделать коммит в одну строку.
12. Отправить сразу 2 файла на внешний репозиторий.
13. На веб интерфейсе создать файл bug_report.xml.
14. Сделать Commit changes (сохранить) изменения на веб интерфейсе.
15. На веб интерфейсе модифицировать файл bug_report.xml, добавить баг репорт в формате XML.
16. Сделать Commit changes (сохранить) изменения на веб интерфейсе.
17. Синхронизировать внешний и локальный репозиторий XML
